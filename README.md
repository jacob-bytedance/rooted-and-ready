## Local Development

To set up and run this project locally:

1. **Prerequisites**
   - Node.js >= 18.17.0
   - pnpm >= 9.14.4

2. **Installation**
   ```bash
   # Install dependencies
   pnpm install

   # Build workspace packages
   pnpm build
   ```

3. **Development**
   ```bash
   # Start the documentation site
   cd apps/docs
   pnpm dev
   ```

   The development server will start at http://localhost:3000
